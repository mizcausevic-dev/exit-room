# Origin

Exit Room exists after the scorecards, board brief, diligence pack, vertical brief, governance registry, deal radar, timing signal index, relationship graph, claims-vs-reality engine, and revenue scorecard.

The goal is to turn that executive-intelligence estate into one explicit exit-readiness and diligence surface:

- which themes can be defended in the room now
- which claims still weaken the board or investor narrative
- where modeled downside remains highest
- what leadership should fix before a process gets harder
